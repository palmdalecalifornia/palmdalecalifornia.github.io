Windows Server 2012 R2 exercises.



The goal is to install Windows Server 2012 R2 from scratch, including VMWare Tools, and create an Active Directory forest.


  • The installation should take around half an hour.
  • We have to install the VMWare Tools to improve performance and to be able to drag and drop between host and guest.
  • The network adapter should be kept as NAT (which is the default setting). The VMs don't need to access the Internet.
  • The server and client VMs will be installed one at a time. Once the server is installed, we can pause the VM and start working on the Windows 7 client.
  • Look at all the steps before installing the VM.
