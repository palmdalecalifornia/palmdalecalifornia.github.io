Creating a Windows 7 Professional VM.



The goal is to create a W7 VM and add it to an Active Directory domain.


  • The installation should take around half an hour.
  • The network adapter should be kept as NAT (which is the default setting). The VMs don't need to access the Internet.
  • We will change the computer name and assign a static IP address.
  • There's no need to have the W2012R2 server up and running yet. Turn it on only when joining the W7 VM to the losangeles.local domain.
  • Look at all the steps before installing the VM. Use the pictures below as guidance.
  • Don't forget to install the VMWare Tools. Same steps as done on the server.
